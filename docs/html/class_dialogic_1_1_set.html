Detailed Description

The Set command is used to create or modify a variable. Variables generally originate in the client environment itself, but can also be created, accessed or modified within Dialogic. If it is important that a global variable is in scope before any chats are run, use the {preload=true} metadata tag, and call ChatRuntime.Preload()
