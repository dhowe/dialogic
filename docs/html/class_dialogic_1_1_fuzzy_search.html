Finds the highest scoring chat which does not violate any of the constraints.

If none match, then start relaxing hard-type constraints until one does.

If all hard constraints have been relaxed and nothing is found, then unrelax hard constraints, lower the staleness threshold, and repeat.

Break ties based on milliseconds since the Chat was last run. If there are still ties, break them with a coin-flip.

Note that the Chat containing the Find object is never returned.
